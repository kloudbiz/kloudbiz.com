1. Generate it locally, do not hand-draw modules. Preferred: Python `segno`
   (`pip install segno` in a scratchpad venv if needed):
   ```python
   import segno
   qr = segno.make('https://kloudbiz.com/#aijoeys', error='m')
   qr.save('qr.svg', kind='svg', xmldecl=False, svgclass=None, lineclass=None,
           dark='#151a2e', light='#ffffff', border=2)
   ```
   Alternatively `npx qrcode -t svg -e M -o qr.svg 'https://kloudbiz.com/#aijoeys'`.

3. **Verify both QRs scan**: screenshot each rendered page (or render the SVGs to PNG)

   same Google Forms URL it does today. Do not ship unverified.
